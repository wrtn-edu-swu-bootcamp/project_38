# FactChecker 서비스 기획안

## 1. 서비스 개요

### 1.1 서비스명
**FactChecker (팩트체커)**

### 1.2 한 줄 소개
가짜뉴스와 허위정보를 학술자료 기반으로 검증하고, 도서관 이용을 활성화하는 팩트체크 서비스

### 1.3 서비스 형태
- 웹 서비스 (데스크톱, 태블릿)
- 모바일 앱 (iOS, Android)

## 2. 기획 배경 및 의도

### 2.1 기획 배경

현대 사회는 정보의 홍수 속에 살아가고 있습니다. 소셜미디어와 인터넷의 발달로 누구나 쉽게 정보를 생산하고 공유할 수 있게 되었지만, 그만큼 가짜뉴스와 허위정보도 빠르게 확산되고 있습니다. 이러한 환경에서 올바른 정보를 식별하는 미디어 리터러시 능력은 현대인의 필수 역량이 되었습니다.

특히 주목할 점은 디지털 네이티브인 **1020세대가 오히려 가짜뉴스에 더 취약**하다는 사실입니다. 유튜브를 통한 뉴스 소비가 2019년 30.8%에서 2022년 63.7%로 급증했으며, 인터넷 이용 시간이 길수록 가짜뉴스 판별 능력이 떨어지는 경향을 보입니다. 유튜브가 가짜뉴스 노출의 가장 주요 경로로 나타나고 있어, 젊은 세대를 위한 팩트체크 도구의 필요성이 더욱 커지고 있습니다.

동시에, AI 기술의 발전은 정보 접근성을 높였지만, AI의 환각(hallucination) 현상으로 인해 AI가 제공하는 정보조차 신뢰하기 어려운 상황이 발생하고 있습니다. 생성형 AI가 만드는 가짜 콘텐츠를 구별하고 알고리즘 편향을 이해하는 능력이 필수가 되었습니다. 사람들은 접한 정보가 사실인지 확인하고 싶어도, 어떻게 검증해야 할지 방법을 모르거나 시간이 부족한 경우가 많습니다.

한편, 도서관은 검증된 지식과 정보를 보유한 공공 지식 인프라임에도 불구하고 이용률이 감소하고 있습니다. 특히 젊은 세대는 도서관보다 인터넷 검색을 선호하며, 도서관이 제공하는 가치를 충분히 활용하지 못하고 있습니다. 정부는 제4차 도서관발전종합계획(2024-2028)을 통해 디지털 전환과 도서관 이용 활성화를 추진하고 있으나, 실질적으로 젊은 세대를 도서관으로 유도할 수 있는 혁신적인 서비스가 필요한 상황입니다.

### 2.2 기획 의도

FactChecker는 이러한 두 가지 사회적 문제를 동시에 해결하고자 기획되었습니다.

#### 의도 1: 미디어 리터러시 향상과 정보 검증 문화 조성

- **가짜뉴스 대응**: 가짜뉴스와 허위정보가 만연한 사회에서 시민들이 스스로 정보의 진위를 판단할 수 있는 도구를 제공합니다.

- **AI 환각 현상 극복**: AI가 제공하는 정보를 맹신하지 않고, 학술논문, 도서, 신뢰할 수 있는 인터넷 자료 등 검증 가능한 출처를 기반으로 정보를 확인할 수 있게 합니다.

- **다양한 입력 방식 지원**: 사용자가 의심스러운 정보를 발견했을 때 즉시 확인할 수 있도록 텍스트 직접 입력, URL 링크, 이미지 업로드 등 다양한 방식으로 정보를 입력받습니다.

- **정도(degree)를 표시하는 검증 방식**: 정보의 사실 여부를 단순히 '참' 또는 '거짓'으로 이분법적으로 판단하는 것이 아니라, 0-100점의 신뢰도 점수와 "사실로 확인됨", "대체로 사실임", "주의가 필요함", "사실이 아님" 등의 단계로 표시합니다. 이는 정보의 복잡성과 맥락을 반영하며, "최근 연구에 의하면 주의가 필요하다"와 같이 구체적인 근거와 함께 제공됩니다.

#### 의도 2: 도서관 이용 활성화와 오프라인 지식 접근성 향상

- **검증 자료와 도서관의 연결**: 정보 검증에 사용된 학술논문, 도서, 인터넷 자료를 사용자에게 직접 보여주되, 특히 도서 자료의 경우 **어느 도서관에서 해당 책을 빌릴 수 있는지**를 함께 안내합니다.

- **공공데이터 활용**: 국립중앙도서관을 비롯한 전국의 공공도서관에서 제공하는 서지 데이터와 소장 정보 공공데이터를 활용하여, 실제 도서관에 책이 소장되어 있는지, 대출이 가능한지 등의 정보를 실시간으로 제공합니다.

- **정보 검증에서 도서관 방문으로 이어지는 선순환**: 단순히 온라인에서 정보를 확인하는 것에 그치지 않고, 더 깊이 있는 정보를 원하는 사용자가 도서관을 직접 방문하여 책을 대출하고 읽을 수 있도록 유도합니다. 이는 곧 도서관 이용 활성화로 이어집니다.

- **지역 기반 도서관 추천**: 사용자의 위치를 기반으로 가까운 도서관을 우선적으로 보여줌으로써, 실제 방문 가능성을 높입니다.

## 3. 시장 및 정책 환경 분석

### 3.1 국내 팩트체크 서비스 현황

#### 3.1.1 주요 서비스 및 운영 상황

**SNU팩트체크**
- 서울대 언론정보연구소가 운영하던 국내 최초의 팩트체크 플랫폼 (2017년 3월 출범)
- 2024년 8월 **재정 고갈로 무기한 운영 중단** (주요 시사점)
- 운영 기간 동안 약 5,000여 건의 팩트체크 검증 수행, 31개 언론사 제휴
- 중단 원인: 여당의 '좌편향' 공격으로 2023년 8월 네이버의 모든 재정지원 중단

**팩트체크넷**
- 한국기자협회·한국PD연합회·방송기자연합회가 2020년 11월 출범
- 크라우드 소싱 방식으로 시민과 기자가 협업하여 검증
- 현재 운영 중

**언론사 팩트체크**
- 뉴스톱, JTBC 팩트체크, SBS 팩트체크, MBC 등 개별 언론사 운영
- 주요 정치적 이슈 및 사회적 쟁점에 대한 검증 수행

#### 3.1.2 현황 분석 및 시사점

**과제:**
- **독립성 유지를 위한 지속적인 재원 확보의 어려움** (SNU팩트체크 사례)
- 정치적 압력에 취약한 구조
- 플랫폼 인지도 및 접근성 부족
- 언론사 중심의 팩트체크는 대중의 직접 참여가 제한적

**FactChecker의 차별화 전략:**
- 도서관 연계를 통한 **공공 가치 창출**로 정치적 중립성 확보
- 학술 자료 기반 검증으로 객관성 강화
- 사용자 중심의 직접 검증 도구 제공 (언론사 검증 기사 의존 탈피)
- 다양한 재원 모델 검토 필요 (공공 재원, 도서관 협력 등)

### 3.2 정책 환경

#### 3.2.1 미디어 리터러시 정책

**국내 정책:**
- 교육부: 초·중·고 교육과정에 '미디어 이해교육' 도입 추진
- 한국언론진흥재단: '뉴스 리터러시교실' 운영
- **AI 리터러시 통합**: 디지털 리터러시 + AI 리터러시 + 시민윤리 결합 형태로 진화 중

**해외 사례:**
- 핀란드: 국가 차원 미디어 교육 의무화, 5년 연속 유럽 미디어 리터러시 조사 최고 수준
- 영국: BBC와 연계한 비판적 사고 훈련 실시

**FactChecker의 교육적 가치:**
- 미디어 리터러시 교육 도구로 활용 가능
- 정보 검증 과정을 통한 학습 효과 제공
- 학교 및 교육기관과의 협력 기회

#### 3.2.2 도서관 정책

**제4차 도서관발전종합계획 (2024-2028):**
- 3대 핵심가치: 따듯한 동행, 공동체성장, 지속가능한 미래
- 4대 정책목표 추진
- **디지털 전환과 신기술 활용 서비스 확대** (FactChecker와 정합성 높음)
- 독서문화 확산과 디지털 문해력 교육 강화
- 지역특화도서관 조성 및 협력플랫폼 구축

**2025년 도서관 정책:**
- 총 8,461억 원 투입
- 'K-도서관 문화랜드마크화' 분야에 5,650억 원 (66.8%) 지원
- 책이음 서비스(지역 간 상호대차 서비스) 활성화

**FactChecker의 정책 연계:**
- 정부의 도서관 디지털 전환 정책과 부합
- 도서관 이용 활성화라는 정책 목표에 기여
- 공공 예산 지원 또는 정책 사업 연계 가능성

### 3.3 기술적 실현 가능성

#### 3.3.1 국립중앙도서관 오픈API (2026년 1월 공개)

**개요:**
- 국립중앙도서관이 2026년 1월 국가서지 Linked Open Data(LOD) 기반 오픈API 서비스 공개
- 행정안전부의 '2025년 국가중점데이터개방지원사업'으로 개발
- **약 3,000만 건의 데이터 개방** (FactChecker 구현의 핵심 인프라)

**제공 데이터 유형:**
1. **서지정보**: 단행본, 연속간행물, 온라인자료 등의 메타데이터
2. **저자정보**: 개인·단체 저자의 정보
3. **주제정보**: 국립중앙도서관 주제명표목표(NLSH) 정보
4. **도서관정보**: 전국 도서관 관련 정보
5. **인터링킹정보**: 국내외 서지·저자·주제데이터간 연결정보

**기술적 장점:**
- 요청 URL: `https://www.nl.go.kr/NL/search/openApi/search.do`
- XML과 JSON 형식 데이터 제공
- **일반인도 쉽게 활용 가능**하도록 설계 (기존 SPARQL 대비 접근성 대폭 향상)
- 실시간 소장정보 및 대출 가능 여부 확인 가능

**FactChecker 적용:**
- 도서 자료 검증 시 실시간 소장 도서관 정보 제공
- 전국 도서관 네트워크 연계
- 지역 기반 도서관 추천 기능 구현
- **2026년 서비스 출시 시 즉시 활용 가능한 인프라 확보**

#### 3.3.2 팩트체크 방법론 진화

**가중 점수 시스템:**
- Media Bias/Fact Check의 2025년 신규 방법론: 최소 10개 헤드라인, 5개 전체 뉴스 기사 검토
- 객관적 지표를 통한 신뢰도 평가
- 인간 검토자가 모든 평가 수행 (AI는 문법 편집에만 제한적 사용)

**AI 기반 검증 프레임워크:**
- VERIFAID: RAG(Retrieval-Augmented Generation) + LLM 활용
- 자동 생성 데이터셋으로 실시간 팩트체크 가능
- 수동 주석 데이터 의존도 감소

**FactChecker 적용:**
- 다층적 신뢰도 평가 시스템 (점수 + 단계)
- AI 보조 검증 + 인간 검토 하이브리드 모델
- 지속적인 검증 정확도 개선 가능

### 3.4 API 활용 계획

FactChecker는 실물 자료 기반 검증을 위해 다양한 공공 API와 서비스 API를 활용합니다.

#### 3.4.1 도서 및 도서관 정보

**도서관 정보나루 API**
- **제공기관**: 국립중앙도서관
- **활용 목적**: 
  - 전국 도서관 소장 자료 검색
  - 도서 서지정보 조회
  - 도서관별 소장 여부 및 대출 가능 여부 확인
- **주요 API**:
  - 소장자료 조회 API
  - 도서관 정보 조회 API
  - ISBN 기반 서지정보 조회

**국립중앙도서관 오픈API (2026년 1월 공개)**
- 3,000만 건 서지정보 데이터
- LOD(Linked Open Data) 기반 통합 검색
- 실시간 소장정보 업데이트

**책이음 서비스 API**
- 전국 공공도서관 통합 대출 정보
- 지역 간 상호대차 정보

#### 3.4.2 학술논문 검색

**Naver 학술정보 API**
- **제공기관**: Naver Developers
- **활용 목적**: 
  - 국내 학술논문 검색
  - 학위논문, 학술지 논문 통합 검색
  - 논문 메타데이터 (제목, 저자, 초록, 출판정보) 수집
- **장점**: 한국어 논문 검색에 최적화

**Google Scholar API (또는 서치 엔진)**
- 국제 학술논문 검색
- 인용 정보 확인
- 다국어 논문 검색

#### 3.4.3 위치 기반 서비스

**Kakao 로컬 API**
- **제공기관**: Kakao Developers
- **활용 목적**:
  - 사용자 위치 기반 가까운 도서관 검색
  - 도서관까지의 거리 계산
  - 지도에 도서관 위치 표시
  - 길찾기 기능 제공

**Kakao 주소 검색 API**
- 도서관 주소 정규화
- 좌표 변환 (주소 → 위경도)

#### 3.4.4 정부 공공데이터

**공공데이터포털 API**
- 각 부처 공식 발표 자료
- 통계청 통계 데이터
- 질병관리청 보건의료 데이터
- 식품의약품안전처 식품안전 정보

- **제공기관**: 누리미디어
- **활용 목적**: 국내 학술논문 원문 접근
- **특징**: KCI 등재 논문 포함, 유료 API

**2. RISS (학술연구정보서비스) API**
- **제공기관**: 한국교육학술정보원
- **활용 목적**: 
  - 국내 학위논문 검색
  - 학술지 논문 검색
  - 무료 API

**3. NDSL (국가과학기술정보센터) API**
- **제공기관**: KISTI
- **활용 목적**: 과학기술 분야 학술논문 및 보고서
- **특징**: 과학기술 분야 특화

**4. PubMed API**
- **제공기관**: 미국 국립보건원 (NIH)
- **활용 목적**: 의학·생명과학 분야 국제 논문
- **특징**: 무료, 영문 논문

**5. Crossref API**
- **제공기관**: Crossref
- **활용 목적**: DOI 기반 논문 메타데이터 조회
- **특징**: 국제 학술논문 출처 검증

**6. 팩트체크 서비스 API (선택적)**
- 뉴스톱, 팩트체크넷 등의 기존 팩트체크 결과 참조
- 중복 검증 방지 및 협력 가능성

**7. 언론진흥재단 빅카인즈 API**
- **제공기관**: 한국언론진흥재단
- **활용 목적**: 국내 주요 언론사 기사 검색 및 분석
- **특징**: 54개 언론사 뉴스 통합 검색

**8. 한국저작권위원회 공공저작물 API**
- 정부 발간 공공저작물 검색
- 정부 보고서, 백서 등 공식 자료

#### 3.4.6 API 활용 시 고려사항

**실물 자료 검증:**
- 모든 API 응답 데이터는 실제 존재하는 자료인지 교차 검증
- ISBN, DOI, 논문 ID 등 고유 식별자를 통한 실물 확인
- 도서관 소장 확인을 통한 실제 존재 여부 검증

**내용 기반 매칭:**
- 단순 키워드 검색이 아닌, 내용 기반 의미론적 검색 (Semantic Search) 적용
- 논문 초록, 도서 목차 등을 분석하여 실제 관련성 확인
- AI 기반 내용 분석으로 검증 대상과의 관련도 평가

**API 사용 한도 및 비용:**
- 무료 API와 유료 API의 적절한 조합
- API 호출 최적화로 비용 절감
- 캐싱 전략으로 반복 호출 방지

## 4. 타겟 사용자

### 4.1 주요 타겟
**일반 대중 (모든 연령대)**

FactChecker는 특정 연령대나 직업군에 한정되지 않고, 정보를 소비하는 모든 사람을 대상으로 합니다.

### 4.2 사용자 페르소나 및 특성 분석

#### 페르소나 1: 정보에 민감한 직장인 (30대, 여성)
- SNS와 뉴스를 통해 다양한 정보를 접함
- 건강, 육아, 생활 정보에 관심이 많음
- 정보가 사실인지 확인하고 싶지만 어떻게 해야 할지 모름
- 시간이 부족해 빠르고 간편한 확인 방법을 원함

#### 페르소나 2: 대학생 및 연구자 (20대, 남성)
- 과제나 연구를 위해 정확한 정보가 필요함
- 학술자료를 찾고 싶지만 어디서 시작해야 할지 막막함
- 도서관에 필요한 책이 있는지 일일이 확인하는 게 번거로움

#### 페르소나 3: 시니어 세대 (60대, 남녀)
- 메신저와 SNS를 통해 받은 정보를 쉽게 믿는 경향이 있음
- 가짜뉴스에 취약하며, 잘못된 정보로 인한 피해를 입을 위험이 있음
- 간단하고 명확한 결과를 원함

#### 페르소나 4: 청소년 및 대학생 (10-20대, 남녀)
- 학습 과정에서 접하는 정보의 진위를 확인해야 함
- **유튜브·SNS를 통한 뉴스 소비 비중이 높아 가짜뉴스 노출 위험 증가**
- 디지털 네이티브임에도 불구하고 가짜뉴스 판별력이 상대적으로 낮음
- 인터넷 이용 시간이 길어질수록 판별 능력 저하 경향
- 미디어 리터러시 교육의 일환으로 사용 가능
- 흥미롭고 직관적인 UI를 선호함

**1020세대 특화 니즈:**
- 유튜브 쇼츠, 릴스 등 짧은 콘텐츠에서 접한 정보의 빠른 검증
- 친구들과 공유하기 전 사실 확인
- 입시 및 취업 정보의 정확성 검증

### 4.3 사용자 니즈
- **진위 확인**: SNS, 뉴스, 메신저 등에서 접한 정보가 사실인지 빠르게 확인하고 싶음
- **신뢰할 수 있는 출처**: 검증 가능한 정보 출처를 알고 싶음
- **학습 및 연구**: 과제, 논문, 프로젝트를 위한 자료를 검증하고 싶음
- **도서관 자료 접근**: 관련 도서가 어느 도서관에 있는지 쉽게 찾고 싶음
- **사용 편의성**: 복잡한 과정 없이 간단하게 사용할 수 있어야 함

## 5. 핵심 기능

### 5.1 다양한 정보 입력 방식

사용자가 다양한 상황에서 즉시 정보를 검증할 수 있도록 세 가지 입력 방식을 제공합니다.

#### 5.1.1 텍스트 직접 입력
- 사용자가 의심스러운 주장이나 정보를 직접 복사하여 입력창에 붙여넣기
- 예: "마늘을 매일 먹으면 암을 예방할 수 있다"
- 짧은 문장부터 긴 단락까지 입력 가능

#### 5.1.2 URL 입력
- 뉴스 기사, 블로그 포스트, SNS 게시물의 링크를 입력
- 시스템이 자동으로 해당 페이지의 내용을 수집하여 주요 주장을 추출
- 전체 기사를 분석하여 여러 주장을 동시에 검증 가능

#### 5.1.3 이미지 업로드
- 사진으로 찍은 텍스트, 스크린샷, 인쇄물 등을 이미지로 업로드
- OCR 기술을 통해 이미지 내의 텍스트를 추출하여 검증
- 예: 단톡방에서 받은 이미지 뉴스, 전단지, 책 내용 등

### 5.2 팩트체크 결과 표시

정보의 복잡성과 맥락을 고려하여, 단순한 참/거짓 이분법이 아닌 다층적 판단을 제공합니다. 이는 국제적으로 검증된 팩트체크 방법론을 참고하여 설계되었습니다.

#### 5.2.1 점수 방식
- **0-100점 신뢰도 점수** 제공
- 점수 범위별 의미:
  - 80-100점: 신뢰도 높음
  - 60-79점: 대체로 신뢰 가능
  - 40-59점: 주의가 필요함
  - 20-39점: 신뢰도 낮음
  - 0-19점: 허위 가능성 높음

#### 5.2.2 단계 방식
명확하고 직관적인 단계 표시:
- **"사실로 확인됨"**: 다수의 신뢰할 수 있는 출처에서 일관되게 확인됨
- **"대체로 사실임"**: 주요 내용은 사실이나 부분적으로 맥락이나 세부사항에서 차이가 있음
- **"주의가 필요함"**: 상반되는 증거가 존재하거나, 일부만 사실임
- **"사실이 아님"**: 명백히 거짓으로 확인됨
- **"판단 불가"**: 검증할 충분한 자료가 없거나, 주관적 의견임

#### 5.2.3 근거 제시
- "최근 연구에 의하면...", "2023년 발표된 논문에 따르면..." 등 구체적인 설명 제공
- 왜 이러한 판단이 내려졌는지 사용자가 이해할 수 있도록 상세한 근거 설명
- 예시:
  > "주의가 필요함 (45점): 비타민 C가 감기를 '완전히 예방'한다는 주장은 과장되었습니다. 2020년 코크란 리뷰에 따르면, 비타민 C는 감기 예방 효과가 제한적이며, 일반인에게는 유의미한 예방 효과가 관찰되지 않았습니다. 다만, 고강도 운동을 하는 사람들에게는 어느 정도 예방 효과가 있을 수 있습니다."

### 5.3 참고 자료 제공

검증 과정에서 활용된 모든 자료를 카테고리별로 정리하여 제공합니다.

**자료 제공 원칙:**
1. **실물 자료만 제공**: 모든 참고 자료는 실제로 존재하는 실물 자료여야 하며, 가상의 자료는 절대 제공하지 않습니다.
2. **내용 기반 관련성**: 단순한 키워드 매칭이 아닌, 실제 내용이 검증 대상과 관련된 자료만 제공합니다.
   - 예: "비타민 섭취가 감기 예방에 효과적이다" 검증 시
   - ✅ 좋은 예: 제목에 "비타민"이 없어도 내용에서 비타민과 감기 예방의 관계를 다룬 논문
   - ❌ 나쁜 예: 제목에 "비타민"만 포함되고 감기와 무관한 "비타민 D와 골다공증" 같은 자료
3. **전문성과 신뢰성**: 검증 가능하고 출처가 명확한 자료만 제공합니다.

#### 5.3.1 학술논문
- **표시 정보**: 논문 제목, 저자, 출판 연도, 학술지명, DOI 링크
- 사용자가 클릭하면 해당 논문의 원문 또는 초록 페이지로 이동
- 예:
  > - **"비타민 C와 감기 예방: 메타분석 연구"**  
  > 저자: Hemilä H, Chalker E  
  > 출판: Cochrane Database Syst Rev, 2020  
  > [DOI: 10.1002/14651858.CD000980.pub4]

#### 5.3.2 도서
- **표시 정보**: 책 제목, 저자, 출판사, 출판 연도, ISBN
- **도서관 소장 정보** (핵심 기능):
  - 해당 도서를 소장하고 있는 도서관 목록
  - 대출 가능 여부
  - 도서관 위치 및 연락처
  - 도서관 홈페이지 링크
- 예:
  > - **"비타민의 진실: 과학적 근거로 보는 영양소"**  
  > 저자: 김영진  
  > 출판: 사이언스북스, 2022  
  > ISBN: 978-89-XXXXX-XX-X  
  > **소장 도서관**:
  >   - 국립중앙도서관 (대출 가능) - 서울특별시 서초구
  >   - 서울시립도서관 (대출 중) - 서울특별시 종로구
  >   - 부산시립중앙도서관 (대출 가능) - 부산광역시 부산진구

#### 5.3.3 신뢰할 수 있는 웹 자료
- **표시 정보**: 제목, 출처(사이트명), URL, 게시 날짜
- 정부 공식 자료, 언론사 팩트체크 기사, 전문 기관 발표 등
- 예:
  > - **"비타민C 섭취와 감기 예방 관련 식약처 공식 입장"**  
  > 출처: 식품의약품안전처  
  > 게시일: 2023.11.15  
  > [링크]

### 5.4 도서관 연계 기능 (핵심 기능)

이 기능은 FactChecker의 가장 차별화된 핵심 기능입니다. **2026년 1월 국립중앙도서관이 공개한 3,000만 건 규모의 오픈API**를 활용하여 실시간 도서관 정보 제공이 가능합니다.

#### 5.4.1 공공데이터 활용
- **국립중앙도서관**: 국가서지 데이터베이스
- **공공도서관**: 전국 공공도서관 통합 서지 정보 시스템
- **대학도서관**: 주요 대학도서관 서지 데이터 (선택적)
- 각 도서관의 공공데이터 포털에서 제공하는 API를 활용하여 실시간 정보 제공

#### 5.4.2 소장 정보 상세 제공
- **소장 도서관 목록**: 해당 도서를 보유한 모든 도서관
- **대출 가능 여부**: 현재 대출 가능한지, 대출 중인지 실시간 확인
- **도서관 정보**:
  - 주소 및 위치
  - 운영 시간
  - 연락처
  - 홈페이지 링크
  - 온라인 예약 링크 (가능한 경우)

#### 5.4.3 지역 기반 추천
- 사용자의 현재 위치 또는 설정한 지역을 기반으로 가까운 도서관을 우선 표시
- 거리순으로 정렬하여 실제 방문 가능성을 높임
- 지도에서 도서관 위치를 직접 확인할 수 있는 기능

#### 5.4.4 도서관 이용 독려
- "이 책을 더 자세히 읽고 싶으신가요? 가까운 도서관에서 대출해보세요!" 등의 메시지
- 도서관 방문을 통한 깊이 있는 학습 권장

## 6. 정보 검증 범위

### 6.1 대상 주제
**전방위 주제를 포괄**

FactChecker는 특정 분야에 한정하지 않고, 사람들이 일상에서 접할 수 있는 모든 주제의 정보를 검증합니다.

- **정치**: 정책, 정치인 발언, 선거 관련 정보
- **사회**: 사건사고, 사회 이슈, 통계 자료
- **과학**: 과학적 주장, 연구 결과, 기술 정보
- **건강**: 의학 정보, 건강식품, 질병 예방법
- **경제**: 경제 지표, 금융 정보, 부동산
- **환경**: 기후변화, 환경 오염, 에너지
- **교육**: 교육 정책, 입시 정보
- **문화**: 역사적 사실, 예술, 문화 현상
- **기타**: 일상생활의 모든 정보

### 6.2 검증 소스

신뢰할 수 있는 다양한 출처를 활용하여 교차 검증합니다.

#### 6.2.1 학술 데이터베이스
- **국내**: KCI (한국학술지인용색인), DBpia, RISS (학술연구정보서비스)
- **국외**: Google Scholar, PubMed, IEEE Xplore, JSTOR
- 동료 심사(peer-review)를 거친 학술논문을 우선 활용

#### 6.2.2 도서 서지 데이터
- 국립중앙도서관 소장 도서
- 공공도서관 소장 도서
- 학술 출판사에서 발행한 전문 서적

#### 6.2.3 정부 공식 자료
- 각 부처 공식 발표

## 8. 기대 효과

### 8.1 사회적 효과

#### 8.1.1 가짜뉴스 및 허위정보 확산 방지
- 개인이 접한 정보를 검증할 수 있는 도구를 제공함으로써, 허위정보의 무분별한 확산을 막을 수 있습니다.
- 특히 SNS와 메신저를 통한 잘못된 정보 공유를 사전에 차단할 수 있습니다.

#### 8.1.2 시민들의 미디어 리터러시 능력 향상
- FactChecker를 사용하는 과정에서 사용자들은 자연스럽게 정보를 비판적으로 바라보는 습관을 기르게 됩니다.
- 어떤 출처가 신뢰할 수 있는지, 왜 이 정보가 사실인지 또는 거짓인지를 학습하게 됩니다.

#### 8.1.3 정보에 대한 비판적 사고 증진
- 모든 정보를 맹신하지 않고, 근거를 확인하는 태도를 형성합니다.
- AI가 제공하는 정보조차 검증이 필요하다는 인식을 확산시킵니다.

#### 8.1.4 사회적 신뢰 회복
- 검증된 정보가 유통됨으로써, 정보에 대한 사회적 신뢰가 회복됩니다.
- 불필요한 논쟁과 갈등을 줄이고, 사실 기반의 건강한 토론 문화를 조성합니다.

### 8.2 도서관 활성화

#### 8.2.1 도서관 이용률 증가
- 정보 검증 과정에서 자연스럽게 도서관 자료를 소개함으로써, 도서관 방문과 대출이 증가합니다.
- 특히 젊은 세대가 도서관을 다시 찾게 되는 계기를 마련합니다.

#### 8.2.2 공공 지식 인프라로서의 도서관 역할 강화
- 도서관이 단순한 책 보관 장소가 아니라, 검증된 지식을 제공하는 신뢰할 수 있는 정보원임을 각인시킵니다.
- 디지털 시대에도 도서관의 가치와 필요성을 재확인시킵니다.

#### 8.2.3 오프라인 독서 문화 활성화
- 온라인 정보 소비에서 오프라인 책 읽기로 이어지는 선순환 구조를 만듭니다.
- 깊이 있는 독서를 통해 더 풍부한 지식을 습득할 수 있도록 유도합니다.

#### 8.2.4 지역 도서관 네트워크 강화
- 전국의 다양한 도서관을 연결함으로써, 공공도서관 네트워크의 가치를 높입니다.
- 각 지역 도서관의 소장 자료를 효과적으로 활용할 수 있게 합니다.

### 8.3 교육적 효과

#### 8.3.1 정보 검증 과정을 통한 학습 효과
- 단순히 결과만 제공하는 것이 아니라, 왜 그러한 판단이 내려졌는지 근거를 보여줌으로써 학습 효과를 제공합니다.
- 사용자는 참고 자료를 직접 확인하며 더 깊이 있는 지식을 습득할 수 있습니다.

#### 8.3.2 신뢰할 수 있는 정보원에 대한 인식 제고
- 학술논문, 정부 공식 자료, 전문 기관 발표 등 어떤 출처가 신뢰할 수 있는지를 자연스럽게 학습합니다.
- 정보의 질을 판단하는 기준을 내재화하게 됩니다.

#### 8.3.3 학술 자료 접근성 향상
- 일반인들에게는 어렵게 느껴지는 학술논문이나 전문 서적에 대한 진입 장벽을 낮춥니다.
- 필요한 학술 자료를 어디서 찾을 수 있는지 안내함으로써, 학술 정보 접근성을 높입니다.

#### 8.3.4 교육 현장에서의 활용
- 학교에서 미디어 리터러시 교육 도구로 활용 가능합니다.
- 학생들이 과제나 프로젝트 수행 시 정보를 검증하는 습관을 기를 수 있습니다.

## 9. 서비스 차별화 포인트

### 9.1 도서관 연계를 통한 독창성
기존의 팩트체크 서비스들은 온라인 정보만 제공하는 데 그치지만, FactChecker는 도서관과 연계하여 오프라인 자료까지 안내합니다. 이는 다음과 같은 차별점을 만듭니다:
- 단순 검증을 넘어 깊이 있는 학습으로 연결
- 공공 인프라(도서관)의 활성화라는 사회적 가치 창출
- 디지털과 아날로그의 균형 있는 조화

### 9.2 이분법을 넘어선 다층적 판단
단순히 '참' 또는 '거짓'이 아니라, 점수와 단계로 정보의 신뢰도를 표현합니다. 이는 정보의 복잡성과 맥락을 존중하며, 사용자에게 더 정확하고 유용한 정보를 제공합니다.

### 9.3 다양한 입력 방식 지원
텍스트, URL, 이미지 등 다양한 방식으로 정보를 입력받아, 사용자의 편의성을 극대화합니다. 특히 이미지 입력 기능은 디지털 약자층도 쉽게 사용할 수 있도록 돕습니다.

### 9.4 전방위 주제 포괄
특정 분야에 국한되지 않고 모든 주제를 다루어, 일상생활의 모든 정보를 검증할 수 있습니다.

### 9.5 기존 팩트체크 서비스와의 차별점

기존의 SNU팩트체크, 팩트체크넷, 언론사 팩트체크 등과 비교하여 FactChecker가 가지는 고유한 차별점:

**1. 사용자 주도형 검증 vs 언론사 주도형**
- 기존: 언론사나 전문 팩트체커가 주요 이슈를 선정하여 검증
- FactChecker: 사용자가 궁금한 정보를 직접 입력하여 즉시 검증

**2. 도서관 연계를 통한 오프라인 확장**
- 기존: 온라인 자료만 제공
- FactChecker: 도서관 소장 정보 제공으로 오프라인 자료까지 연결

**3. 정치적 중립성 확보**
- 기존: 정치적 압력에 취약 (SNU팩트체크 사례)
- FactChecker: 공공 인프라(도서관) 연계를 통한 공익성 강조로 중립성 확보

**4. 지속가능한 운영 모델**
- 기존: 재정 확보 어려움으로 운영 중단 사례 발생
- FactChecker: 도서관 정책과 연계한 공공 지원, 교육기관 협력 등 다각적 재원 모델

## 10. 지속가능성 및 향후 계획

### 10.1 지속가능한 운영 전략

SNU팩트체크의 운영 중단 사례가 시사하듯, 팩트체크 서비스의 지속가능성은 매우 중요한 과제입니다. FactChecker는 다음과 같은 전략으로 지속가능성을 확보합니다.

#### 10.1.1 다각적 재원 확보
- **공공 재원**: 도서관 정책 예산, 미디어 리터러시 교육 예산과 연계
- **도서관 협력**: 국립중앙도서관, 지역 공공도서관과의 협력 사업
- **교육기관 연계**: 학교 미디어 리터러시 교육 도구로 활용
- **정부 지원 사업**: 문화체육관광부, 교육부 등의 정책 사업 참여

#### 10.1.2 정치적 중립성 유지
- 도서관 활성화라는 비정치적 공공 가치 강조
- 학술 자료 중심의 객관적 검증 방식
- 다양한 관점의 자료를 균형있게 제시

#### 10.1.3 커뮤니티 기반 성장
- 사용자 참여형 검증 요청 시스템
- 교육 기관, 도서관, 시민단체와의 네트워크 구축
- 오픈소스 기반 개발로 투명성 확보 (선택적)

### 10.2 별도 작성 예정 문서

#### 와이어프레임 (UI/UX 설계)
- 사용자 인터페이스 디자인
- 화면 구성 및 사용자 흐름(User Flow)
- 각 기능별 상세 화면 설계

#### 디자인 가이드
- 브랜딩 (로고, 슬로건)
- 컬러 팔레트
- 타이포그래피
- 아이콘 및 일러스트레이션 스타일
- UI 컴포넌트 가이드

### 10.3 기술 문서
기술적인 구현 방법, 시스템 아키텍처, 사용 기술 스택 등은 별도의 기술 문서로 작성 예정입니다. 현재 기획안에서는 서비스의 '무엇'과 '왜'에 집중하며, '어떻게'는 추후 다루게 됩니다.

## 11. 결론

FactChecker는 단순한 팩트체크 서비스를 넘어, 사회적 문제 해결과 공공 가치 창출을 목표로 하는 서비스입니다.

**첫째**, 가짜뉴스와 허위정보가 만연한 시대에 시민들이 스스로 정보를 검증할 수 있는 도구를 제공하여, 미디어 리터러시를 향상시키고 건강한 정보 생태계를 조성합니다.

**둘째**, 도서관이라는 공공 인프라와 연계하여, 정보 검증에서 도서관 방문으로 이어지는 선순환을 만들어냅니다. 이는 도서관 이용 활성화라는 사회적 효과를 가져옵니다.

**셋째**, 학술논문, 도서, 정부 공식 자료 등 신뢰할 수 있는 출처를 기반으로 정보를 검증함으로써, AI 환각 현상이 만연한 시대에 진정으로 믿을 수 있는 정보 검증 수단을 제공합니다.

FactChecker는 모든 연령대의 일반 대중을 대상으로 하며, 텍스트, URL, 이미지 등 다양한 방식으로 정보를 입력받아 편의성을 극대화합니다. 검증 결과는 점수와 단계로 표시하여 정보의 복잡성을 반영하며, 상세한 근거와 참고 자료를 함께 제공합니다.

특히, 검증에 사용된 도서가 어느 도서관에 소장되어 있는지를 안내하는 것은 FactChecker만의 가장 차별화된 핵심 기능입니다. 이를 통해 사용자는 온라인에서의 정보 확인을 넘어, 도서관을 방문하여 더 깊이 있는 지식을 탐구할 수 있습니다.

FactChecker는 정보의 진위를 가리는 것뿐만 아니라, 사람들이 정보를 대하는 태도를 변화시키고, 도서관이라는 소중한 공공 자산을 재발견하게 하며, 궁극적으로는 더 건강하고 신뢰할 수 있는 정보 사회를 만드는 데 기여할 것입니다.
