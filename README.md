## 🔑 데이터 소스 전략 (현실적 접근)

### 현재 보유 중 ✅

**즉시 사용 가능한 API 3개:**
- 도서관 정보나루 API ⭐ (국립중앙도서관 소장 자료)
- Kakao Developers API ⭐ (도서관 위치, 지도)
- Naver Developers API ⭐ (학술정보, 책, 뉴스)

**API 키 불필요한 공개 소스:**
- Google Scholar (전 세계 학술논문)
- PubMed (의학 논문, 키 없이도 기본 사용 가능)
- RISS 웹 검색 (국내 학위논문)
- 알라딘 API (즉시 발급 가능, 5분)
- 국회도서관, 교보문고 등 (웹 크롤링)

### 데이터 충분성 분석

| 데이터 유형 | 커버리지 | 주요 소스 |
|------------|---------|----------|
| 국내 학술논문 | 80% | Naver API + RISS 웹 |
| 해외 학술논문 | 90% | Google Scholar 크롤링 |
| 의학 논문 | 85% | PubMed 웹 |
| 도서 정보 | 95% | Naver + 정보나루 + 알라딘 |
| 도서관 위치 | 100% | 정보나루 + Kakao |
| 팩트체크 기사 | 90% | Naver 뉴스 |

**결론**: 현재 보유한 리소스만으로도 충분히 서비스 구현 가능! ✅

### 즉시 시작하기

1. **알라딘 TTB Key 발급** (5분, 즉시 가능)
   - https://www.aladin.co.kr/ttb/wblog_main.aspx
   - 회원가입 후 바로 발급

2. **보유 API 테스트**
   - Naver 학술정보/책/뉴스 검색
   - 도서관 정보나루 도서 검색
   - Kakao 도서관 위치 검색

3. **웹 크롤링 준비**
   - Google Scholar
   - RISS, PubMed

자세한 내용은 **[데이터 소스 전략](docs/data-source-strategy.md)** 참조

## 🎯 서비스 핵심 기능

### 1. 다양한 정보 입력
- 텍스트 직접 입력
- URL 링크 입력
- 이미지 업로드

### 2. 다층적 팩트체크
- 0-100점 신뢰도 점수
- 5단계 판정 (사실로 확인됨 ~ 사실이 아님)
- 상세한 근거 제시

### 3. 참고 자료 제공
- 학술논문 (DOI 링크)
- 도서 (도서관 소장 정보 포함) ⭐
- 정부 공식 자료
- 언론사 팩트체크 기사

### 4. 도서관 연계 (핵심 차별화)
- 전국 도서관 소장 정보 실시간 제공
- 대출 가능 여부 확인
- 지역 기반 가까운 도서관 추천
- 도서관 방문 독려

## 🌟 서비스 차별점

### 기존 팩트체크 서비스와의 차이

| 특징 | 기존 서비스 | FactChecker |
|------|------------|-------------|
| 검증 방식 | 언론사 선정 이슈 | 사용자 직접 입력 |
| 자료 범위 | 온라인만 | 온라인 + 도서관 |
| 도서 정보 | 없음 | 소장 도서관 안내 ⭐ |
| 정치적 중립성 | 취약 | 도서관 연계로 공익성 강화 |
| 지속가능성 | 재정 불안정 | 다각적 재원 모델 |

### 주요 차별화 포인트

1. **도서관 연계**: 검증 자료가 있는 도서관 위치 안내 → 도서관 이용 활성화
2. **실물 자료 중심**: 가상이 아닌 실제 존재하는 자료만 제공
3. **내용 기반 매칭**: 단순 키워드가 아닌 실제 내용의 관련성으로 자료 선정
4. **정도 표시**: 이분법이 아닌 점수+단계로 신뢰도 표현
